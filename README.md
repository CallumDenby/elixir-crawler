# Crawler

Source Code: `/lib`
Unit Tests: `/lib`
Config: `/config/config.exs`

## Requirements

Elixir 1.16.1 / OTP 26

To install dependencies:

```
mix deps.get
```

## Running

To run in a non-interactive environment:

```
mix run --no-halt
```

To run in an interactive repl:

```
iex -S mix
```

## Tests

Located in `/test`

To run the unit tests:

```
mix test
```
